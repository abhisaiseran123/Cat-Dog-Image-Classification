# Cat and Dog Image Classification using CNN

##  Project Overview

This project implements a **Cat and Dog Image Classification** model using **Deep Learning**. A **Convolutional Neural Network (CNN)** was built and trained using TensorFlow and Keras to classify images as either a cat or a dog.

The model was trained on a labeled dataset after applying image preprocessing techniques such as resizing, normalization, and data augmentation to improve its performance and reduce overfitting. After training, the model was saved to **Google Drive** using Google Colab's mounting feature, allowing it to be reused later without retraining.

---

##  Features

- Binary image classification (Cat vs Dog)
- Built using Convolutional Neural Networks (CNN)
- Image preprocessing and normalization
- Data augmentation to improve generalization
- Model saved for future inference without retraining
- Developed in Google Colab
